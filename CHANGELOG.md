# 更新日志

## v1.2.1 (2026-04-28)

### 新增
- Ctrl+左键/右键点击 http 链接自动调用默认浏览器打开
- 编辑器右键上下文菜单（撤销/重做/剪切/复制/粘贴/全选）
- 菜单栏剪切/复制按钮根据选区状态动态启用/禁用
- 右键上下文菜单剪切/复制按钮根据选区状态动态启用/禁用

### 修复
- 双击 .md 文件打开后内容为空（渲染进程 IPC 就绪时序问题）
- 首次双击文件打开时 `process.argv` 未解析
- 深色主题选中文字样式不可见
- 构建产物命名规范化：`项目名_版本_版本号`

## v1.1.0 (2026-04-27)

### 新增
- 表格内联控件：表格边缘直接显示 `+`/`−` 按钮，无需工具栏即可增减行列
- 表格左上角 `×` 按钮一键删除整表
- 自动保存草稿功能（失焦时将内容保存到 draft.md）
- 覆盖层方式实现表格控件，不影响编辑器核心 DOM

### 修复
- 窗口位置/大小从未从配置中恢复（始终用默认值）
- 打开文件后标题栏错误显示"已修改"标记
- Windows Chrome 路径 `\x86` 被误解析为十六进制转义
- 关于对话框版本号硬编码为 v1.0.0
- 新建文件时 `clearContent` 触发伪造的"已修改"状态
- 应用图标仅含单个 256px 尺寸，现含 16~256 共 7 种尺寸

### 优化
- 大纲重建从每次按键改为 requestAnimationFrame 节流
- TurndownService 实例从每次保存新建改为模块级复用
- openFile 中 setInterval 泄漏问题修复
- 状态栏和打印样式从 light.css 移至 base.css
- 移除工具栏冗余的 `+行`/`-行`/`+列`/`-列` 文字按钮
- 移除隐式全局变量和死代码（invokeWithTimeout）
- 列删除按钮从仅查 `<th>` 改为查第一行所有单元格

### 工程
- 构建后自动打包 win-unpacked 为 zip
- 图标生成脚本支持多尺寸 ICO 输出
