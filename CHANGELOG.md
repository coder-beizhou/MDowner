## v2.7.0 (2026-07-01) —「行云流水」

### ⚡ 性能优化（大文档连续输入更流畅）

- 大纲：加结构签名早返回（标题结构未变则跳过重建）+ `#outline` 事件委托一次绑定

- 表格控件：光标不在表格且 overlay 已空时早返回；overlay 点击改委托

### 🎨 代码块语言选择器

- 复用 `setNodeMarkup({language})` 机制，切换后高亮即时更新、保存往返保留

### 📋 原生菜单人性化
- 文件 → 最近文件 子菜单（读 `config.recentFiles`，打开文件后自动刷新）
- 文件 → 新建 Markdown / 新建 JSON / 新建 YAML

- 编辑 → 格式 子菜单（加粗/斜体/删除线/行内代码/正文/标题1~6/列表/引用/代码块，带快捷键）
- 编辑 → 插入 子菜单（分割线/代码块/表格/链接/图片）

### 🐛 修复

- 修复 `setFileContent` 对含 `"type":"doc"` 但非有效 JSON 的文件解析失败后无回退、编辑器空白的问题

## v2.3.2 (2026-04-30) —「铁壁防线」

### 🛡️ 文件加载全链路加固
v2.3.1 存在致命缺陷——拖入/恢复非 Markdown 文件会直接卡死整个应用。根因是 ZIP/exe 等二进制文件被当作文本读入，传给 `marked.parse()` 导致渲染进程 OOM。现在三层防护：
- **主进程拦截**：`openFile()` 检查扩展名（仅 `.md`/`.markdown`/`.txt`），文件超过 5MB 直接拒绝
- **恢复标签拦截**：`restoreTabs()` 过滤掉非 Markdown 扩展名的历史标签，并清理损坏的 config
- **拖拽拦截**：拖入多个文件时逐个过滤，跳过非 Markdown 文件

### 🚦 关闭逻辑修复
- 主进程 `close` 事件之前无条件 `preventDefault`，如果渲染进程已经卡死，窗口永远关不掉。现在通过 `hasUnsavedTabs` 标记判断——只有确实有未保存内容时才阻止关闭
- 渲染进程崩溃自动检测：`render-process-gone` 事件监听，崩溃后不阻塞关闭

### 🧹 草稿生命周期管理
- 关闭标签时立即删除对应草稿文件（`draft_{tabId}.md`），不再堆积
- 启动时自动清理超过 7 天的孤儿草稿文件
- 累计清理了 41 个历史遗留草稿（14MB+）

### 🔧 其他修复
- 修复 `.claude/settings.local.json` 被反复提交到仓库的问题（更新 `.gitignore`）
- 移除残留的调试日志

## v2.0.1 (2026-04-29) —「多标签觉醒」

### 🗂️ 多标签页
这是 MDowner 出生以来最大的一次架构升级——从单文件编辑器变成了多标签页编辑器。

- **标签栏**：工具栏下方多了标签栏，`+` 新建标签、点击切换、`×` / 中键关闭
- **快捷键**：`Ctrl+T` 新建、`Ctrl+Tab` / `Ctrl+Shift+Tab` 切换、`Ctrl+W` 关闭
- **独立状态**：每个标签有自己的内容、光标位置、撤销历史、修改标记
- **独立草稿**：每个标签有独立草稿文件，不会互相覆盖
- **持久化**：关闭软件再打开，上次的所有标签和当前选中标签完美恢复
- **批量关闭**：关闭窗口时弹出逐个选择对话框，可自由勾选要保存的标签
- **文件丢失保护**：恢复标签时文件已不存在则自动跳过并清理记录
- **拖拽打开**：拖多个文件到窗口，每个文件各占一个标签
- **启动优化**：多标签恢复时并行读文件 + 批量创建，不逐个切换

### 🏷️ 代码块语言标签重构
- 标签从 JS overlay 像素计算改为直接渲染在 `<pre>` 内 CSS 锚定——缩放窗口不再漂移
- 标签大小固定，不随窗口变化

### 🚫 代码块不再顶头
- ProseMirror 插件守护，代码块永远不是文档第一个节点

### 🔧 修复与优化
- 关闭标签时弹出保存确认对话框（不再偷偷自动保存）
- 标签名过长时修改蓝点不再被截断
- 窗口每次打开固定 1200×800 居中
- 启动时大纲自动加载
- 标签栏右键菜单：关闭标签 / 关闭其他 / 关闭已保存 / 关闭所有
- 右键菜单不再触发编辑器左键行为

## v1.5.1 (2026-04-29) —「高亮觉醒」

### 🎨 代码块高亮重构
说到做到，代码高亮从"意思意思亮了就行"升级为"认认真真给你好看"。

- **语言全家桶**：从区区 16 种一口气拉到 39 种，Rust、Swift、Kotlin、Dart、Elixir、Haskell……你写啥它都能认
- **自动识破**：没标语言的代码块也能自己猜出来（`highlightAuto`），还带智商在线——如果第一行是注释或者废话，自动跳过看后面的真代码
- **标签不骗人了**：之前右上角永远冷冰冰一个 `text`，现在老老实实显示 `python`、`javascript`……原因是 TipTap 把语言类名藏在 `<code>` 上，之前却在 `<pre>` 上找，鸡同鸭讲

### 🚫 代码块再也不顶头了
文档第一个节点是代码块？那你就永远没法在它上面写东西。现在加了守护插件——一旦有代码块想当第一名，立刻塞个空段落上去，不跟你商量。

### 👁️ 表格按钮按需显隐
表格四周的 `+` / `−` / `×` 按钮不再全天候待命。光标踏进表格时才现身，走出去就消失。干干净净，不挡视野。

### 🔧 小修小补
- 语言标签 `language-objective-c` 这种带连字符的类名终于能正确解析了
- 文件打开时立即触发语言检测，之前等 300ms 像在发呆
- 合并重复 import，去尾逗号
